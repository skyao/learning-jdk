# JDK学习笔记

### 内容介绍

JDK全称Java SE Development kit(JDK)，即java标准版开发包，它提供编译，运行java程序所需要的各种工具和资源，包括java编译器，java运行时环境，以及常用的java类库等。

### 访问方式

这是个人学习 JDK 的笔记，请点击下面的链接阅读:
